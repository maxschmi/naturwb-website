# /Resultate.shp
################
Eine Shape-Datei namens "Resultate.shp", die das NatUrWB-Ergebnis auf Ebene der Modellgebiete wiedergibt.
Die Projektion der Datei ist dabei ETRS89 / UTM Zone 32N (EPSG:25832).

Spalten:
---------
 - nat_id:     Die ID der Naurraumeinheit
 - sim_id:     Die ID des Modellgebiets
 - gen_id:     Die ID der Bodengesellschaft in der BÜK Sachdatenbank
 - Boden_kurz: Kurzbeschreibung der Bodengesellschaft
 - Boden_lang: lange Beschreibung der Bodengesellschaft als Text
 - color:      HEX-Farbcode der Bodengesellschaft
 - nat_name:   Name der Naturraumeinheit
 - N:          Niederschlag in mm/a
 - kap.A:      der kapillare Aufstieg in mm/a
 - ET:         die Evapotranspiration in mm/a
 - Abfluss:    der Abfluss, inklusive dem Zwischenabfluss, dies ist demnach die Summe aus OA und ZA, in mm/a
 - OA:         der Oberflächenabfluss in mm/a
 - ZA:         der Zwischenabfluss in mm/a
 - GWNB:       die Grundwasserneubildung in mm/a
